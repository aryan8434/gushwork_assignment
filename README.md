# Gushwork Assignment - HDPE Pipes Website

A responsive, single-page website for Mangalam HDPE Pipes with sticky header, carousel functionality, and a product portfolio.

## Features

- Sticky header that appears and disappears on scroll
- Responsive image carousel with navigation controls and wrap-around looping
- Mobile-first layout with breakpoints at 1400px, 1200px, 1080px, 800px, 360px, and 328px
- Portfolio section with distinct product images
- Product dropdown with six textile machinery products
- Contact forms with submit handling
- Overflow protection for very small screens

## Getting Started

1. Open `index.html` in a web browser.
2. No build process or dependencies are required.
3. All styling and scripting are included in the project files.
