# Gushwork Assignment - HDPE Pipes Website

A responsive, single-page website for Mangalam HDPE Pipes with sticky header, carousel functionality, and comprehensive product portfolio.

## Features

- **Sticky Header**: Dynamically appears/disappears on scroll
- **Image Carousel**: Responsive carousel with navigation controls and 360-degree looping
- **Responsive Design**: Mobile-first approach with breakpoints at 1400px, 1200px, 1080px, 800px, 360px, and 328px
- **Portfolio Section**: Three-card layout with distinct product images (hero.png, img1.jpg, img2.jpg)
- **Product Dropdown**: Navigation dropdown with six textile machinery products
- **CTA Forms**: Contact forms with form submission handling
- **Mobile Optimization**: Proper spacing and overflow handling for screens below 328px

## File Structure

```
.

```
